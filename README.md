# 百分之一小作文生成器

AstrBot 插件 —— 在 QQ 聊天中通过关键词触发，自动生成符合 TapTap《百分之一》活动格式的游戏推荐帖。

## 功能特性

- **关键词触发**：发送「小作文」或「我的百分之一」随机选择游戏生成，或指定游戏名生成
- **联网搜索**：自动搜索游戏资料，让 AI 生成更贴合实际的内容
- **频率限制**：基于 QQ 号的全局频率限制，防止滥用
- **黑名单管理**：管理员可针对特定群聊/私聊开启或关闭功能
- **WebUI 管理**：通过 AstrBot 插件管理页面可视化管理黑名单

## 使用方式

### 用户指令

| 触发方式 | 说明 |
|---------|------|
| `小作文` | 随机选择预设游戏生成小作文 |
| `小作文 原神` | 指定游戏生成小作文 |
| `我的百分之一` | 同「小作文」 |
| `我的百分之一 原神` | 同「小作文 原神」 |

### 管理员指令

| 指令 | 说明 |
|------|------|
| `/开启小作文功能` | 为当前会话开启功能 |
| `/关闭小作文功能` | 为当前会话关闭功能 |
| `/小作文模型测试` | 测试 AI 模型连接是否正常 |
| `/小作文搜索测试 游戏名` | 测试联网搜索功能 |

## 配置说明

在 AstrBot 插件管理页面中配置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 管理员QQ号列表 | 支持多个管理员 | 空 |
| AI Base URL | API 地址，兼容 OpenAI 格式 | - |
| AI API Key | API 密钥 | - |
| AI Model | 模型名称 | deepseek-chat |
| 预设游戏列表 | 随机生成时的游戏池 | 见配置 |
| 频率限制时间窗口 | 滑动窗口（分钟） | 10 |
| 窗口内最大次数 | 每 QQ 号窗口内限制 | 3 |
| 每日最大次数 | 每 QQ 号每日限制 | 20 |
| 是否启用联网搜索 | 搜索游戏资料 | 是 |
| 搜索超时时间 | 毫秒 | 8000 |
| 默认是否启用 | 新会话默认状态 | 是 |

## 安装方式

1. 将本插件目录放入 AstrBot 的插件目录中
2. 在 AstrBot 管理页面启用插件
3. 配置 AI 模型参数（Base URL、API Key、Model）
4. 配置管理员 QQ 号
